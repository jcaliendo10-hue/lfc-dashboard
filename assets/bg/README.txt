PER-TAB BACKGROUND IMAGES
=========================

Each tab can show a faded HD image behind its content. Drop a .jpg here named
exactly after the tab, and it appears automatically (faded to ~12% opacity).
If a file is missing, that tab simply shows no background — nothing breaks.

Filenames the dashboard looks for (assets/bg/<name>.jpg):

  overview.jpg    — Anfield exterior / stadium hero shot
  squad.jpg       — team photo or squad line-up
  compare.jpg     — two players / head-to-head imagery
  depth.jpg       — squad / tactics board
  lineup.jpg      — Anfield pitch from the stands
  simulator.jpg   — transfer desk / FSG imagery
  fit.jpg         — Iraola on the touchline
  transfers.jpg   — the boardroom / negotiation room
  news.jpg        — press room / media wall
  injuries.jpg    — the AXA Training Centre medical / gym
  worldcup.jpg    — World Cup 2026 imagery
  analytics.jpg   — data / training-ground analysis
  hierarchy.jpg   — the AXA Training Centre (training ground)

Tips:
- Use landscape HD images (1600px+ wide) for the best look.
- They're heavily faded, so high-contrast images still read fine.
- Only add images you have the right to use. This is an unofficial fan project
  and is not affiliated with or endorsed by Liverpool FC.
