OFFICIAL LIVERPOOL FC LOGO
==========================

The dashboard is wired to display an official club logo from this folder.

To use the official crest:
  1. Save the official Liverpool FC logo image as:  lfc-logo.png
     (a square PNG with a transparent background works best, e.g. 200x200+)
  2. Place it in this folder:  assets/lfc-logo.png
  3. Commit & push. The header and footer will automatically use it.

Until that file exists, a generic red/gold "LFC" badge is shown as a fallback,
so the dashboard never looks broken.
